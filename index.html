Bones

Version 0.1

About Bones?

Bones is a fork from Skeleton, the small collection of well-organized CSS files that can help you rapidly develop sites that look beautiful at any size, be it a 17" laptop screen or an iPhone. It's based on a responsive grid, but also provides very basic CSS for typography, buttons, forms and media queries. Go ahead, resize this super basic page to see the grid in action. Bones borns to improve Skeleton features adding support for Full HD monitors and using LESS for better styling.

Three Core Principles

Bones is built on three core principles:

  • A Responsive Grid Down To Mobile: Elegant scaling from a browser to tablets to mobile.
  • Fast to Start: It's a tool for rapid development with best practices
  • Style Agnostic: It provides the most basic, beautiful styles, but is meant to be overwritten.

Docs & Support

The easiest way to really get started with Bones is to check out the full docs and info from Skeleton at www.getskeleton.com.. Bones is also open-source and has a project on git (Sekeleton original project), so check that out if you want to report bugs or create a pull request.
